# SereneFlow Landing Page

A premium yoga sanctuary offering Vinyasa, Hatha, and Power yoga — in studio and online. Find your flow with SereneFlow.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
